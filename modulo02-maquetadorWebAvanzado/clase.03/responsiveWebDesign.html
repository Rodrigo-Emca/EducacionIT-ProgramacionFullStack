Responsive Web Design

Mobile First

  • Fluid Design

    Utiliza porcentajes para definir el tamaño de los ejementos, permitiendo que estos se adapten a su contenedor, pero en pantallas grande los mismos se estiran demasiado.


    Tambien existe una variable llamada Liquid Design que establece el uso de máximos y mínomos, pero esto genera espacios demasiado grandes esntre elementos.

  • Adaptive Desgin

    Consiste en diseñar una página para cada tipo de dispositivo en el que se visualice el sitio, pero en este caso, los contenidos se repiten y son independientes.


    Esta propuesta es ideal para clientes con bajo presupuesto, ya que generalmente se utilizan 2 versiones del sitio principal (desktop) y se utilizan herramientas para el redireccionamiento.

  • Responsive Design

    Diseño utilizando multimedia flexible, un sistema de frillas adaptable e implementa el concepto one web for all para evitar la duplicación de contenido.
