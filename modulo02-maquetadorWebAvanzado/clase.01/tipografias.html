Google fonts

Google nos proporciona tipografias para poder usarlas en nuestros sitios web. Son gratuitas y no existen problemas de licencias con ellas.

Pueden encontrarse aqui: Google Fonts

Se recomienda, por legibilidad y accesibilidad, usar tipografias de las familias: Serif y Sans Serif.

Una vez elegida la familia, saldrán una variedad de tipografias para usar.

Finalmente, hay que elegir el "peso" o grosor, y las voy "agregando" con el botón + que aparece al lado de cada tipografia.

Como máximo, se recomiendan hasta 3 tipografias y 3 pesos por cada una.

Recordar que mientras más tipografias, mas peso le agregamos a la página. Por eso, se recomienda que cuanto menos tipografias, mejor.

Se copia el link de Google fonts, y se lo importa en el html, arriba del linkeo del css. El html primero debe leer la tipografia y luego, el css. Una vez importada la tipografia, la usamos en el css.

La otra forma es importando la tipografia directamente al css.

body {
font-family: 'Chakra Petch', sans-serif;
}

Tener en cuenta: google fonts trabaja con internet. Si el usuario se queda sin internet, mientras esta recorriendo la página, no podrá ver las tipografias cargadas.

Para ello existe un salvaconducto: se suele agregar alguna tipografia local, cuando cargo el font-family, como un "Times New Roman". Esto hace que cuando el usuario se queda sin internet, no pierda todo el texto, sino que simplemente se cambia la letra.
