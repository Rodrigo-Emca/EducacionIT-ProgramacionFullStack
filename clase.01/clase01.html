Introducción

En la clase de hoy, se presentan aspectos iniciales para la programación web.

Aspectos importantes:

  • Instalación de Visual Studio Code

    • Si bien existen otros editores de texto, como Brackets, Sublime Text y Atom, usaremos preferente VisualStudio Code.

  • Editores Online:

    • Tambien existen editores Online, como Plunker y CodePen. Estos no necesitan instalación en la pc, pero requieren conexión a internet.
