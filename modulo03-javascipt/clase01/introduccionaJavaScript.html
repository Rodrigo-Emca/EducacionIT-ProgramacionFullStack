Introducción a JavaScript

¿Qué es un algoritmo?

Es una serie de pasos, instrucciones, para lograr algo o llevar a cabo una tarea.

Por ejemplo: abrir la puerta.

Pero la computadora no tiene la capacidad de abstraer, esto implica que le voy a tener que indicar paso a paso todo para que pueda resolver la tarea.

Código binario

Lenguaje de 0 y 1.

Puede usarse como: 0 - apagado 1 - prendido

O configurar nuestros mensajes: 0 - nublado 1 - soleado

Configurar el archivo JavaScript

Para que el archivo JavaScript funcione, hay que linkear el html con el archivo js, tal como lo haciamos con el css.

Para ello, se puede linkear el archivo, en el head del archivo, pero se recomienda agregarlo al final del body, justo antes del cierre de esta etiqueta.

El console.log

El console.log se usa para mostrar en consola, algo que se escribe en el js. Se trata de una palabra reservada.

Tambien existe console.error(imprime en color rojo) y console.warn (imprime en color amarrillo).
