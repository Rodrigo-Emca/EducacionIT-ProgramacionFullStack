¿Qué es un algoritmo?

Es una serie de pasos, instrucciones, para lograr algo o llevar a cabo una tarea.

Por ejemplo: abrir la puerta.

Pero la computadora no tiene la capacidad de abstraer, esto implica que le voy a tener que indicar paso a paso todo para que pueda resolver la tarea.

Código binario

Lenguaje de 0 y 1.

Puede usarse como: 0 - apagado 1 - prendido

O configurar nuestros mensajes: 0 - nublado 1 - soleado
