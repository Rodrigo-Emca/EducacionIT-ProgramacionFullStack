Formularios en HTML

Para crear un formulario, utilizamos la etiqueta "form"

Atributos de la etiqueta form

  • action: Es la ruta (url) a donde se va a enviar la información de recopilación.
  • method: (protocolo http => metodos http) - Método GET - POST. Solo a traves de estas dos formas puedo inviar la información. Es cómo se va a enviar la información. Get => A través de la URL o POST => a traves del cuerpo del HTML
  • target: El mismo comportamiento que tenian en los links (anchor tags). Si le coloco target="_blank" se abre en otra pestaña. Si le coloco target="_self" se abre en la misma pestaña.

Formulario al enviarse va a armar la URL (Query Sting)

Cuando envio la informacion, a traves del metodo GET. El formulario automaticamente arma la URL para enviar la informacion. A esa URL se la llama Query Sting.
